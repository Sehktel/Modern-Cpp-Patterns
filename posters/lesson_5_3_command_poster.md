# 📊 Плакат: Command Pattern (Команда)

## 🎯 Назначение Command Pattern

```
┌─────────────────────────────────────────────────────────────────┐
│                    COMMAND PATTERN                              │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│  📦 ИНКАПСУЛИРУЕТ ЗАПРОС КАК ОБЪЕКТ                           │
│  🔄 ПОДДЕРЖИВАЕТ ОПЕРАЦИИ UNDO/REDO                            │
│  📋 ПОЗВОЛЯЕТ СТРОИТЬ ОЧЕРЕДИ КОМАНД                           │
│  🎯 РАЗДЕЛЯЕТ ОТПРАВИТЕЛЯ И ПОЛУЧАТЕЛЯ                         │
└─────────────────────────────────────────────────────────────────┘

Применение:
┌─────────────────┬───────────────────────────────────────────────┐
│   GUI Systems   │ • Кнопки и меню                              │
├─────────────────┼───────────────────────────────────────────────┤
│   Undo/Redo     │ • Отмена и повтор операций                   │
├─────────────────┼───────────────────────────────────────────────┤
│   Macro         │ • Запись и воспроизведение команд             │
├─────────────────┼───────────────────────────────────────────────┤
│   Logging       │ • Логирование операций                        │
├─────────────────┼───────────────────────────────────────────────┤
│   Queuing       │ • Очереди команд                              │
└─────────────────┴───────────────────────────────────────────────┘
```

## 🔄 Проблема без Command

```
┌─────────────────────────────────────────────────────────────────┐
│                    ПРОБЛЕМА: ЖЕСТКАЯ СВЯЗЬ                     │
└─────────────────────────────────────────────────────────────────┘

❌ ПЛОХОЙ ДИЗАЙН - Нарушение принципов:
┌─────────────────────────────────────────────────────────────────┐
│                    Button (Нарушает принципы)                  │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │              Button                                        ││
│  │                                                             ││
│  │  void onClick() {                                          ││
│  │      switch(buttonType) {                                  ││
│  │          case COPY:                                        ││
│  │              // Логика копирования                          ││
│  │              document.copy();                              ││
│  │              clipboard.set(document.getSelection());       ││
│  │              logger.log("Copy operation");                 ││
│  │              break;                                        ││
│  │                                                             ││
│  │          case PASTE:                                       ││
│  │              // Логика вставки                              ││
│  │              document.paste();                             ││
│  │              document.insert(clipboard.get());             ││
│  │              logger.log("Paste operation");                ││
│  │              break;                                        ││
│  │                                                             ││
│  │          case DELETE:                                      ││
│  │              // Логика удаления                             ││
│  │              document.delete();                            ││
│  │              logger.log("Delete operation");               ││
│  │              break;                                        ││
│  │                                                             ││
│  │          // ❌ НОВАЯ КОМАНДА ТРЕБУЕТ МОДИФИКАЦИИ!         ││
│  │      }                                                      ││
│  │  }                                                          ││
│  └─────────────────────────────────────────────────────────────┘│
│                                                                 │
│  Проблемы:                                                     │
│  ❌ Нарушение Open/Closed Principle                            │
│  ❌ Сложно добавлять новые команды                             │
│  ❌ Нет поддержки Undo/Redo                                    │
│  ❌ Сложно тестировать отдельные операции                      │
└─────────────────────────────────────────────────────────────────┘
```

## ✅ Решение с Command

```
┌─────────────────────────────────────────────────────────────────┐
│                    РЕШЕНИЕ: COMMAND PATTERN                   │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                    ХОРОШИЙ ДИЗАЙН                             │
│                                                                 │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐        │
│  │   Copy      │    │   Paste     │    │   Delete    │        │
│  │  Command   │    │  Command    │    │  Command    │        │
│  │             │    │             │    │             │        │
│  │ • execute()│    │ • execute() │    │ • execute() │        │
│  │ • undo()   │    │ • undo()    │    │ • undo()    │        │
│  └─────────────┘    └─────────────┘    └─────────────┘        │
│         ▲                 ▲                 ▲                  │
│         │                 │                 │                  │
│  ┌──────┴─────────────────┴─────────────────┴──────────┐      │
│  │              Command                                 │      │
│  │                                                     │      │
│  │  virtual void execute() = 0;                        │      │
│  │  virtual void undo() = 0;                           │      │
│  │  virtual std::string getDescription() = 0;          │      │
│  └─────────────────────────────────────────────────────┘      │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │              Button                                        ││
│  │                                                             ││
│  │  std::unique_ptr<Command> command;                         ││
│  │                                                             ││
│  │  void setCommand(std::unique_ptr<Command> cmd) {           ││
│  │      command = std::move(cmd);                              ││
│  │  }                                                          ││
│  │                                                             ││
│  │  void onClick() {                                          ││
│  │      if (command) {                                        ││
│  │          command->execute();                               ││
│  │      }                                                      ││
│  │  }                                                          ││
│  └─────────────────────────────────────────────────────────────┘│
│                                                                 │
│  Преимущества:                                                  │
│  ✅ Соблюдение Open/Closed Principle                           │
│  ✅ Легко добавлять новые команды                             │
│  ✅ Поддержка Undo/Redo                                        │
│  ✅ Легко тестировать отдельные операции                       │
└─────────────────────────────────────────────────────────────────┘
```

## 🎨 Варианты реализации

### 1. Классическая реализация
```
┌─────────────────────────────────────────────────────────────────┐
│                  КЛАССИЧЕСКАЯ РЕАЛИЗАЦИЯ                      │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ // Абстрактная команда                                         │
│ class Command {                                                │
│ public:                                                         │
│     virtual ~Command() = default;                              │
│     virtual void execute() = 0;                                │
│     virtual void undo() = 0;                                   │
│     virtual std::string getDescription() = 0;                 │
│ };                                                              │
│                                                                 │
│ // Конкретная команда - копирование                            │
│ class CopyCommand : public Command {                           │
│ private:                                                        │
│     Document& document;                                         │
│     std::string copiedText;                                     │
│                                                                 │
│ public:                                                         │
│     CopyCommand(Document& doc) : document(doc) {}              │
│                                                                 │
│     void execute() override {                                  │
│         copiedText = document.getSelection();                  │
│         document.copy();                                        │
│     }                                                           │
│                                                                 │
│     void undo() override {                                      │
│         // Копирование не имеет undo                            │
│     }                                                           │
│                                                                 │
│     std::string getDescription() override {                    │
│         return "Copy";                                          │
│     }                                                           │
│                                                                 │
│     std::string getCopiedText() const { return copiedText; }    │
│ };                                                              │
│                                                                 │
│ // Конкретная команда - вставка                                │
│ class PasteCommand : public Command {                           │
│ private:                                                        │
│     Document& document;                                         │
│     std::string pastedText;                                     │
│     size_t position;                                            │
│                                                                 │
│ public:                                                         │
│     PasteCommand(Document& doc) : document(doc) {}              │
│                                                                 │
│     void execute() override {                                  │
│         position = document.getCursorPosition();                │
│         pastedText = document.getClipboard();                   │
│         document.paste();                                      │
│     }                                                           │
│                                                                 │
│     void undo() override {                                      │
│         document.setCursorPosition(position);                  │
│         document.deleteText(pastedText.length());              │
│     }                                                           │
│                                                                 │
│     std::string getDescription() override {                    │
│         return "Paste";                                         │
│     }                                                           │
│ };                                                              │
└─────────────────────────────────────────────────────────────────┘
```

### 2. Реализация с макросами
```
┌─────────────────────────────────────────────────────────────────┐
│              РЕАЛИЗАЦИЯ С МАКРОСАМИ                           │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ class MacroCommand : public Command {                          │
│ private:                                                        │
│     std::vector<std::unique_ptr<Command>> commands;            │
│                                                                 │
│ public:                                                         │
│     void addCommand(std::unique_ptr<Command> cmd) {            │
│         commands.push_back(std::move(cmd));                    │
│     }                                                           │
│                                                                 │
│     void execute() override {                                  │
│         for (auto& cmd : commands) {                          │
│             cmd->execute();                                    │
│         }                                                       │
│     }                                                           │
│                                                                 │
│     void undo() override {                                     │
│         // Отменяем команды в обратном порядке                 │
│         for (auto it = commands.rbegin(); it != commands.rend(); ++it) {│
│             (*it)->undo();                                     │
│         }                                                       │
│     }                                                           │
│                                                                 │
│     std::string getDescription() override {                    │
│         std::string desc = "Macro: ";                          │
│         for (const auto& cmd : commands) {                     │
│             desc += cmd->getDescription() + " ";               │
│         }                                                       │
│         return desc;                                            │
│     }                                                           │
│ };                                                              │
│                                                                 │
│ // Использование                                               │
│ auto macro = std::make_unique<MacroCommand>();                  │
│ macro->addCommand(std::make_unique<CopyCommand>(document));    │
│ macro->addCommand(std::make_unique<PasteCommand>(document));    │
│ macro->execute(); // Выполняет все команды                     │
│ macro->undo();    // Отменяет все команды                      │
└─────────────────────────────────────────────────────────────────┘
```

### 3. Реализация с лямбдами
```
┌─────────────────────────────────────────────────────────────────┐
│              РЕАЛИЗАЦИЯ С ЛЯМБДАМИ                            │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ class LambdaCommand : public Command {                         │
│ private:                                                        │
│     std::function<void()> executeFunc;                          │
│     std::function<void()> undoFunc;                             │
│     std::string description;                                    │
│                                                                 │
│ public:                                                         │
│     LambdaCommand(std::function<void()> exec,                  │
│                   std::function<void()> undo,                   │
│                   const std::string& desc)                     │
│         : executeFunc(exec), undoFunc(undo), description(desc) {}│
│                                                                 │
│     void execute() override {                                  │
│         if (executeFunc) {                                     │
│             executeFunc();                                      │
│         }                                                       │
│     }                                                           │
│                                                                 │
│     void undo() override {                                      │
│         if (undoFunc) {                                        │
│             undoFunc();                                         │
│         }                                                       │
│     }                                                           │
│                                                                 │
│     std::string getDescription() override {                    │
│         return description;                                    │
│     }                                                           │
│ };                                                              │
│                                                                 │
│ // Использование                                               │
│ auto command = std::make_unique<LambdaCommand>(                │
│     []() { document.copy(); },                                 │
│     []() { /* undo logic */ },                                 │
│     "Copy"                                                      │
│ );                                                              │
└─────────────────────────────────────────────────────────────────┘
```

```
│     auto command = std::make_unique<CopyCommand>(document);     │
│     command->execute();                                         │
│                                                                 │
│     EXPECT_EQ("Hello", command->getCopiedText());             │

│ }                                                               │
└─────────────────────────────────────────────────────────────────┘
```

## 💡 Практические рекомендации

```
┌─────────────────────────────────────────────────────────────────┐
│                  ПРАКТИЧЕСКИЕ РЕКОМЕНДАЦИИ                     │
└─────────────────────────────────────────────────────────────────┘

1. 🎯 Выбор реализации:
   • Классическая - для общих случаев
   • Макросы - для сложных операций
   • Лямбды - для простых случаев
   • std::function - для гибкости

2. 🔧 Принципы проектирования:
   • Слабая связанность между отправителем и получателем
   • Абстракция через интерфейсы
   • Инкапсуляция запросов
   • Разделение ответственности

3. 🧪 Тестирование:
   • Тестируйте каждую команду отдельно
   • Используйте моки для изоляции тестов
   • Проверяйте корректность undo/redo
   • Тестируйте макросы

4. ⚠️ Типичные ошибки:
   • Забывание реализации undo()
   • Неправильное управление состоянием
   • Игнорирование исключений
   • Создание слишком много команд

5. 📚 Лучшие практики:
   • Используйте RAII для управления командами
   • Применяйте move semantics
   • Документируйте контракты команд
   • Рассмотрите асинхронные команды

6. 🚀 Оптимизация:
   • Кэшируйте часто используемые команды
   • Используйте move semantics
   • Рассмотрите batch команды
   • Профилируйте критические участки
```
