# 📊 Плакат: Facade Pattern (Фасад)

## 🎯 Назначение Facade Pattern

```
┌─────────────────────────────────────────────────────────────────┐
│                    FACADE PATTERN                               │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│  🏢 ПРЕДОСТАВЛЯЕТ УПРОЩЕННЫЙ ИНТЕРФЕЙС К СЛОЖНОЙ ПОДСИСТЕМЕ   │
│  🔧 СКРЫВАЕТ СЛОЖНОСТЬ ВНУТРЕННЕЙ РЕАЛИЗАЦИИ                   │
│  🚀 УПРОЩАЕТ ВЗАИМОДЕЙСТВИЕ С СИСТЕМОЙ                        │
│  🛡️ СОБЛЮДЕНИЕ PRINCIPLE OF LEAST KNOWLEDGE                   │
└─────────────────────────────────────────────────────────────────┘

Применение:
┌─────────────────┬───────────────────────────────────────────────┐
│   Database      │ • Упрощение работы с БД                      │
├─────────────────┼───────────────────────────────────────────────┤
│   File Systems  │ • Упрощение работы с файлами                 │
├─────────────────┼───────────────────────────────────────────────┤
│   Web Services  │ • Упрощение API                              │
├─────────────────┼───────────────────────────────────────────────┤
│   Game Engines  │ • Упрощение игровых систем                    │
├─────────────────┼───────────────────────────────────────────────┤
│   Compilers     │ • Упрощение компиляции                        │
└─────────────────┴───────────────────────────────────────────────┘
```

## 🔄 Проблема без Facade

```
┌─────────────────────────────────────────────────────────────────┐
│                    ПРОБЛЕМА: СЛОЖНОСТЬ СИСТЕМЫ                 │
└─────────────────────────────────────────────────────────────────┘

❌ ПЛОХОЙ ДИЗАЙН - Нарушение принципов:
┌─────────────────────────────────────────────────────────────────┐
│                    Client (Нарушает принципы)                  │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │              Client                                        ││
│  │                                                             ││
│  │  void processOrder(Order order) {                          ││
│  │      // 1. Проверка инвентаря                              ││
│  │      InventoryManager inventory;                           ││
│  │      if (!inventory.checkAvailability(order)) {           ││
│  │          throw std::runtime_error("Out of stock");         ││
│  │      }                                                      ││
│  │                                                             ││
│  │      // 2. Обработка платежа                               ││
│  │      PaymentProcessor payment;                             ││
│  │      if (!payment.processPayment(order.getAmount())) {     ││
│  │          throw std::runtime_error("Payment failed");        ││
│  │      }                                                      ││
│  │                                                             ││
│  │      // 3. Отправка заказа                                  ││
│  │      ShippingService shipping;                             ││
│  │      shipping.shipOrder(order);                            ││
│  │                                                             ││
│  │      // 4. Уведомление клиента                             ││
│  │      NotificationService notification;                     ││
│  │      notification.sendConfirmation(order);                 ││
│  │                                                             ││
│  │      // 5. Логирование                                      ││
│  │      Logger logger;                                         ││
│  │      logger.logOrder(order);                               ││
│  │                                                             ││
│  │      // ❌ КЛИЕНТ ЗНАЕТ СЛИШКОМ МНОГО!                     ││
│  │  }                                                          ││
│  └─────────────────────────────────────────────────────────────┘│
│                                                                 │
│  Проблемы:                                                     │
│  ❌ Нарушение Principle of Least Knowledge                     │
│  ❌ Сложность использования                                   │
│  ❌ Тесная связанность с подсистемой                           │
│  ❌ Сложность тестирования                                     │
└─────────────────────────────────────────────────────────────────┘
```

## ✅ Решение с Facade

```
┌─────────────────────────────────────────────────────────────────┐
│                    РЕШЕНИЕ: FACADE PATTERN                     │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                    ХОРОШИЙ ДИЗАЙН                             │
│                                                                 │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐        │
│  │ Inventory   │    │  Payment    │    │ Shipping    │        │
│  │ Manager     │    │ Processor   │    │ Service     │        │
│  │             │    │             │    │             │        │
│  │ • check()   │    │ • process() │    │ • ship()    │        │
│  │ • reserve() │    │ • validate()│    │ • track()   │        │
│  └─────────────┘    └─────────────┘    └─────────────┘        │
│         ▲                 ▲                 ▲                  │
│         │                 │                 │                  │
│  ┌──────┴─────────────────┴─────────────────┴──────────┐      │
│  │              OrderFacade                              │      │
│  │                                                     │      │
│  │  InventoryManager inventory;                        │      │
│  │  PaymentProcessor payment;                          │      │
│  │  ShippingService shipping;                          │      │
│  │  NotificationService notification;                  │      │
│  │  Logger logger;                                     │      │
│  │                                                     │      │
│  │  void processOrder(Order order) {                  │      │
│  │      // Упрощенный интерфейс                         │      │
│  │      if (!inventory.checkAvailability(order)) {    │      │
│  │          throw std::runtime_error("Out of stock");   │      │
│  │      }                                              │      │
│  │                                                     │      │
│  │      if (!payment.processPayment(order.getAmount())) {│      │
│  │          throw std::runtime_error("Payment failed");│      │
│  │      }                                              │      │
│  │                                                     │      │
│  │      shipping.shipOrder(order);                    │      │
│  │      notification.sendConfirmation(order);          │      │
│  │      logger.logOrder(order);                       │      │
│  │  }                                                  │      │
│  └─────────────────────────────────────────────────────┘      │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │              Client                                        ││
│  │                                                             ││
│  │  OrderFacade facade;                                        ││
│  │                                                             ││
│  │  void processOrder(Order order) {                          ││
│  │      facade.processOrder(order);                           ││
│  │      // ✅ ПРОСТОЙ И ПОНЯТНЫЙ ИНТЕРФЕЙС!                  ││
│  │  }                                                          ││
│  └─────────────────────────────────────────────────────────────┘│
│                                                                 │
│  Преимущества:                                                  │
│  ✅ Соблюдение Principle of Least Knowledge                    │
│  ✅ Упрощение использования                                    │
│  ✅ Слабая связанность с подсистемой                           │
│  ✅ Легкость тестирования                                       │
└─────────────────────────────────────────────────────────────────┘
```

## 🏗️ Структура Facade Pattern

```
┌─────────────────────────────────────────────────────────────────┐
│                    СТРУКТУРА ПАТТЕРНА                          │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                    Facade (Фасад)                              │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │                    Facade                                 ││
│  │                                                             ││
│  │  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐        ││
│  │  │ Subsystem   │  │ Subsystem   │  │ Subsystem   │        ││
│  │  │     A       │  │     B       │  │     C       │        ││
│  │  │             │  │             │  │             │        ││
│  │  │ • method1() │  │ • method1() │  │ • method1() │        ││
│  │  │ • method2() │  │ • method2() │  │ • method2() │        ││
│  │  └─────────────┘  └─────────────┘  └─────────────┘        ││
│  │         ▲                 ▲                 ▲              ││
│  │         │                 │                 │              ││
│  │  ┌──────┴─────────────────┴─────────────────┴──────────┐  ││
│  │  │              Facade                                │  ││
│  │  │                                                     │  ││
│  │  │  SubsystemA subsystemA;                             │  ││
│  │  │  SubsystemB subsystemB;                             │  ││
│  │  │  SubsystemC subsystemC;                             │  ││
│  │  │                                                     │  ││
│  │  │  void simplifiedOperation() {                      │  ││
│  │  │      subsystemA.method1();                          │  ││
│  │  │      subsystemB.method1();                          │  ││
│  │  │      subsystemC.method1();                          │  ││
│  │  │  }                                                  │  ││
│  │  └─────────────────────────────────────────────────────┘  ││
│  └─────────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                    Client (Клиент)                             │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │                    Client                                  ││
│  │                                                             ││
│  │  ┌─────────────┐                                           ││
│  │  │ Facade      │                                           ││
│  │  │             │                                           ││
│  │  │ • operation │                                           ││
│  │  └─────────────┘                                           ││
│  │         ▲                                                   ││
│  │         │                                                   ││
│  │  ┌──────┴──────────────────────────────────────────────┐  ││
│  │  │              Client                                 │  ││
│  │  │                                                     │  ││
│  │  │  Facade facade;                                     │  ││
│  │  │                                                     │  ││
│  │  │  void useSystem() {                                │  ││
│  │  │      facade.simplifiedOperation();                 │  ││
│  │  │  }                                                  │  ││
│  │  └─────────────────────────────────────────────────────┘  ││
│  └─────────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────────┘
```

```
│         │                              ▲                      │
│         │                              │                      │
│         │      new Facade()            │                      │

```

## 🎨 Варианты реализации

### 1. Классическая реализация
```
┌─────────────────────────────────────────────────────────────────┐
│                  КЛАССИЧЕСКАЯ РЕАЛИЗАЦИЯ                      │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ // Подсистемы                                                   │
│ class InventoryManager {                                        │
│ public:                                                          │
│     bool checkAvailability(const Order& order) {                │
│         // Проверка наличия товара                              │
│         return true;                                             │
│     }                                                            │
│                                                                  │
│     void reserveItems(const Order& order) {                     │
│         // Резервирование товара                                │
│     }                                                            │
│ };                                                               │
│                                                                  │
│ class PaymentProcessor {                                         │
│ public:                                                          │
│     bool processPayment(double amount) {                        │
│         // Обработка платежа                                    │
│         return true;                                             │
│     }                                                            │
│                                                                  │
│     bool validatePayment(double amount) {                       │
│         // Валидация платежа                                    │
│         return true;                                             │
│     }                                                            │
│ };                                                               │
│                                                                  │
│ class ShippingService {                                          │
│ public:                                                          │
│     void shipOrder(const Order& order) {                        │
│         // Отправка заказа                                       │
│     }                                                            │
│                                                                  │
│     std::string trackOrder(const Order& order) {               │
│         // Отслеживание заказа                                   │
│         return "shipped";                                        │
│     }                                                            │
│ };                                                               │
│                                                                  │
│ // Фасад                                                         │
│ class OrderFacade {                                              │
│ private:                                                         │
│     InventoryManager inventory;                                  │
│     PaymentProcessor payment;                                    │
│     ShippingService shipping;                                    │
│                                                                  │
│ public:                                                          │
│     void processOrder(const Order& order) {                     │
│         // Упрощенный интерфейс                                 │
│         if (!inventory.checkAvailability(order)) {             │
│             throw std::runtime_error("Out of stock");            │
│         }                                                        │
│                                                                  │
│         if (!payment.processPayment(order.getAmount())) {       │
│             throw std::runtime_error("Payment failed");          │
│         }                                                        │
│                                                                  │
│         inventory.reserveItems(order);                           │
│         shipping.shipOrder(order);                               │
│     }                                                            │
│                                                                  │
│     std::string trackOrder(const Order& order) {                │
│         return shipping.trackOrder(order);                      │
│     }                                                            │
│ };                                                               │
└─────────────────────────────────────────────────────────────────┘
```

### 2. Реализация с шаблонами
```
┌─────────────────────────────────────────────────────────────────┐
│              РЕАЛИЗАЦИЯ С ШАБЛОНАМИ                           │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ template<typename... Subsystems>                               │
│ class Facade {                                                  │
│ private:                                                        │
│     std::tuple<Subsystems...> subsystems;                      │
│                                                                 │
│ public:                                                         │
│     Facade(Subsystems... subs) : subsystems(subs...) {}        │
│                                                                 │
│     template<size_t Index>                                      │
│     auto& getSubsystem() {                                      │
│         return std::get<Index>(subsystems);                    │
│     }                                                           │
│                                                                 │
│     void simplifiedOperation() {                                │
│         // Использование подсистем через getSubsystem<Index>() │
│     }                                                           │
│ };                                                              │
│                                                                 │
│ // Использование                                                │
│ auto facade = Facade<InventoryManager, PaymentProcessor,       │
│                      ShippingService>(                          │
│     InventoryManager{},                                         │
│     PaymentProcessor{},                                         │
│     ShippingService{}                                           │
│ );                                                              │
└─────────────────────────────────────────────────────────────────┘
```

### 3. Реализация с лямбдами
```
┌─────────────────────────────────────────────────────────────────┐
│              РЕАЛИЗАЦИЯ С ЛЯМБДАМИ                            │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│ class LambdaFacade {                                            │
│ private:                                                        │
│     std::function<void()> operation;                            │
│                                                                 │
│ public:                                                         │
│     LambdaFacade(std::function<void()> op) : operation(op) {}  │
│                                                                 │
│     void execute() {                                            │
│         operation();                                            │
│     }                                                           │
│ };                                                              │
│                                                                 │
│ // Использование                                                │
│ auto facade = LambdaFacade([]() {                               │
│     InventoryManager inventory;                                 │
│     PaymentProcessor payment;                                   │
│     ShippingService shipping;                                   │
│                                                                 │
│     // Упрощенная операция                                      │
│     inventory.checkAvailability(order);                         │
│     payment.processPayment(amount);                             │
│     shipping.shipOrder(order);                                  │
│ });                                                             │
└─────────────────────────────────────────────────────────────────┘
```

## 🎯 Когда использовать Facade

```
┌─────────────────────────────────────────────────────────────────┐
│                КОГДА ИСПОЛЬЗОВАТЬ FACADE                      │
└─────────────────────────────────────────────────────────────────┘

✅ ХОРОШИЕ СЛУЧАИ:
┌─────────────────┬───────────────────────────────────────────────┐
│ Database        │ • Упрощение работы с БД                      │
│                 │ • Скрытие сложности SQL                       │
│                 │ • Упрощение транзакций                       │
├─────────────────┼───────────────────────────────────────────────┤
│ File Systems    │ • Упрощение работы с файлами                 │
│                 │ • Скрытие сложности файловой системы          │
│                 │ • Упрощение операций ввода/вывода            │
├─────────────────┼───────────────────────────────────────────────┤
│ Web Services    │ • Упрощение API                              │
│                 │ • Скрытие сложности протоколов                │
│                 │ • Упрощение аутентификации                    │
├─────────────────┼───────────────────────────────────────────────┤
│ Game Engines    │ • Упрощение игровых систем                    │
│                 │ • Скрытие сложности рендеринга               │
│                 │ • Упрощение управления ресурсами              │
├─────────────────┼───────────────────────────────────────────────┤
│ Compilers       │ • Упрощение компиляции                        │
│                 │ • Скрытие сложности парсинга                  │
│                 │ • Упрощение оптимизации                       │
└─────────────────┴───────────────────────────────────────────────┘

❌ ПЛОХИЕ СЛУЧАИ:
┌─────────────────┬───────────────────────────────────────────────┐
│ Простые случаи  │ • Когда подсистема не сложная                │
│                 │ • Когда нет необходимости в упрощении        │
├─────────────────┼───────────────────────────────────────────────┤
│ Производительность│ • Когда критична производительность          │
│                 │ • Когда накладные расходы недопустимы         │
├─────────────────┼───────────────────────────────────────────────┤
│ Сложность       │ • Когда усложнение не оправдано              │
│                 │ • Когда можно использовать простые решения  │
└─────────────────┴───────────────────────────────────────────────┘
```

## 💡 Практические рекомендации

```
┌─────────────────────────────────────────────────────────────────┐
│                  ПРАКТИЧЕСКИЕ РЕКОМЕНДАЦИИ                     │
└─────────────────────────────────────────────────────────────────┘

1. 🎯 Выбор реализации:
   • Классическая - для общих случаев
   • Шаблоны - для производительности
   • Лямбды - для простых случаев
   • std::function - для гибкости

2. 🔧 Принципы проектирования:
   • Слабая связанность между клиентом и подсистемой
   • Абстракция через интерфейсы
   • Инкапсуляция сложности
   • Разделение ответственности

3. 🧪 Тестирование:
   • Тестируйте фасад как единое целое
   • Используйте моки для изоляции тестов
   • Проверяйте корректность упрощенного интерфейса
   • Тестируйте обработку ошибок

4. ⚠️ Типичные ошибки:
   • Создание слишком сложного фасада
   • Игнорирование производительности
   • Неправильное управление состоянием
   • Отсутствие валидации входных данных

5. 📚 Лучшие практики:
   • Используйте RAII для управления ресурсами
   • Применяйте move semantics
   • Документируйте упрощенный интерфейс
   • Рассмотрите асинхронные операции

6. 🚀 Оптимизация:
   • Кэшируйте часто используемые операции
   • Используйте move semantics
   • Рассмотрите lazy loading
   • Профилируйте критические участки
```
